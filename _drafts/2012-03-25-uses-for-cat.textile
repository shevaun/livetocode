h2. Append line(s) to end of a file

bc. $ cat >> notes.txt
Get milk on the way home 
Ctrl-D

h2. Create a new file

bc. $ cat > awesome_script.rb
#!/usr/bin/env ruby
puts "Hello World!"
Ctrl-D

h2. Count occurrences of a string

bc. $ cat <path_to_files> | grep -c <string>
